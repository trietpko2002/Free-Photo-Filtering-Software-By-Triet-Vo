Anti-Freeze Hàng Loạt

Anti-Freeze Batch

Phân loại, Copy/Cut hàng ngàn file RAW cùng lúc bằng phím Enter. Giao diện không bao giờ bị đơ (Not Responding).

Sort, Copy/Cut thousands of RAW files instantly via Enter key. UI remains perfectly responsive without freezing.

Chụp Tethering (Live-Shoot)

Tethering (Live-Shoot)

Theo dõi thư mục máy ảnh thời gian thực. Ảnh vừa chụp lập tức nạp lên phần mềm để kiểm tra và xử lý ngay.

Real-time camera folder monitoring. Newly captured photos are auto-loaded for instant review and processing.

Đổi Tên Auto (Ảnh Thẻ)

Smart Rename (ID Photo)

Nhập tên khách hàng (VD: NguyenVanA), xuất ảnh phần mềm tự động đổi tên và đánh số thứ tự thông minh.

Enter client name (e.g., JohnDoe), the software auto-renames and numbers exported files intelligently.

Cảnh Báo Trùng Lặp

Duplicate Warning

Ngăn chặn việc xuất nhầm 1 file 2 lần vào cùng thư mục. Cho phép tùy chọn Bỏ qua hoặc Tạo bản sao.

Prevents accidentally exporting the same file twice to a folder. Options to Skip or Create a duplicate.

Cập Nhật OTA (Auto-Update)

OTA Auto-Update

Phần mềm tự động kiểm tra phiên bản trên GitHub, tải về và ghi đè âm thầm. Luôn có bản mới nhất chỉ với 1-click.

Automatically checks GitHub for updates, downloads, and replaces the exe silently. Always stay up-to-date.

App Lock & Safe Eject

App Lock & Safe Eject

Khóa bảo vệ dự án hình ảnh nhạy cảm bằng Mật khẩu. Hỗ trợ ngắt kết nối thẻ SD an toàn trực tiếp trên giao diện.

Password-protect sensitive projects. Support safe removal of SD cards directly from the software UI.

**Photo Filter Free** là giải pháp phần mềm Windows dạng Portable (Chạy ngay không cần cài đặt) được thiết kế đặc biệt dành cho **Nhiếp ảnh gia và Editor**. Giúp bạn phân loại, chọn lọc và backup hàng vạn tấm ảnh RAW/JPEG từ thẻ nhớ vào máy tính với tốc độ cực hạn mà không bao giờ bị treo máy.

---

## 🇻🇳 TIẾNG VIỆT

### ✨ Tính năng Đột phá
* **Thuật toán Anti-Freeze (Chống Nghẽn):** Xử lý Batch (hàng loạt) hàng ngàn file dung lượng lớn cùng lúc mà giao diện vẫn mượt mà.
* **Bộ lọc Nhiếp ảnh (Deep Filters):** Chỉ lọc Ảnh (tự động bỏ qua Video), tự động loại bỏ các file rác/thumbnail dựa trên dung lượng (KB).
* **Bảo mật App Lock:** Đặt mật khẩu khóa thư viện ảnh nhạy cảm/private của khách hàng.
* **Đọc thông số máy ảnh (EXIF):** Tự động hiển thị Model máy ảnh, Kích thước pixel và Ngày giờ bấm máy ngay trên phần mềm.
* **An toàn Dữ liệu:** Tính năng **Safe Eject** giúp ngắt kết nối Thẻ nhớ/SD Card an toàn tuyệt đối trước khi rút.

### 📥 Cách Sử Dụng (Portable)
1. Truy cập trang **[Releases](LINK_RELEASES_CUA_BAN)**.
2. Tải xuống tệp `PhotoFilterFree.exe` mới nhất.
3. Cắm thẻ nhớ vào máy tính, click đúp vào tệp `.exe` và bắt đầu lọc ảnh ngay!

### ⌨️ Cẩm nang Phím tắt
* `C / X`: Chuyển đổi giữa chế độ **COPY** (Backup thẻ) và **CUT** (Dọn sạch thẻ).
* `1 - 5`: Nhấn để chọn nhanh 5 thư mục đích đã cài đặt (VD: 1=Tiệc, 2=Cô Dâu...).
* `Enter`: Thực thi chuyển ảnh.
* `Ctrl/Shift + Click`: Chọn nhiều ảnh cùng lúc.

## 🇺🇸 ENGLISH

### ✨ Photography Features
* **Anti-Freeze Batch Processing:** Handle thousands of heavy RAW/JPEG files simultaneously without UI freezing.
* **Deep Filters:** Automatically skip video files and small junk/thumbnail files based on KB size.
* **App Lock:** Password-protect the application to secure sensitive/private client photos.
* **EXIF Metadata:** Instantly view Camera Model, Resolution, and Original Shot Date.
* **Safe Eject:** Safely disconnect SD Cards/USB drives directly from the app to prevent data corruption.

### 📥 How to Use (Portable Version)
1. Navigate to the **[Releases](https://github.com/trietpko2002/web_gioi_thieu_phan_mem_loc_anh_v8/releases/tag/stable)** page.
2. Download the latest `PhotoFilterFree.exe` file.
3. Plug in your SD card, double-click the `.exe`, and start sorting!

### ⌨️ Hotkeys Guide
* `C / X`: Toggle between **COPY** (Backup) and **CUT** (Clear card) modes.
* `1 - 5`: Quick-select target folders.
* `Enter`: Execute the file transfer.
* `Ctrl/Shift + Click`: Select multiple photos at once.

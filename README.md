## 📖 Giới Thiệu Phần Mềm (Introduction)

**Photo Filter Studio** là một phần mềm mã nguồn mở (100% Miễn phí) được thiết kế đặc biệt dành cho các Nhiếp ảnh gia, Thợ chụp sự kiện, và các Studio chụp ảnh thẻ / hồ sơ. 

**❓ Phần mềm này giải quyết vấn đề gì?**
Sau mỗi buổi chụp sự kiện, phóng sự cưới hay kỷ yếu, thợ ảnh thường phải đối mặt với hàng ngàn, thậm chí hàng chục ngàn bức ảnh (bao gồm cả file RAW và Video rất nặng). Việc xem lại (Culling) và chép file sang các thư mục phân loại bằng File Explorer mặc định của Windows cực kỳ chậm chạp, dễ bị đơ máy (Not Responding) và tốn rất nhiều thời gian.

**💡 Giải pháp từ Photo Filter Studio:**
Phần mềm mang đến một quy trình làm việc (Workflow) khép kín, tập trung vào **Tốc độ** và **Sự ổn định**. Với công nghệ "Anti-Freeze" và bộ phím tắt tối ưu, bạn có thể lướt xem ảnh tốc độ cao và phân loại hàng trăm file chỉ trong chớp mắt mà không lo giật lag. Ở phiên bản Studio, phần mềm còn đóng vai trò như một trạm thu nhận ảnh trực tiếp từ máy ảnh (Live-Shoot), giúp thợ ảnh thẻ đổi tên và trả file cho khách chỉ trong vài giây.

## 🔥 Tính Năng Nổi Bật (Highlight Features)

* ⬇️ **Tự Động Cập Nhật (Auto-Update OTA):** Tích hợp công nghệ tải trực tiếp bản cập nhật `.exe` mới nhất từ GitHub Releases. Hệ thống tự động ghi đè file và khởi động lại phần mềm mượt mà, giúp người dùng luôn có phiên bản tốt nhất mà không cần tải thủ công.
* 📸 **Chế độ Chụp Liên Kết (Live-Shoot / Tethering):** Phần mềm tự động giám sát thư mục máy ảnh kết nối. Ngay khi bạn bấm máy, ảnh sẽ tự động được tải lên phần mềm và hiển thị ngay lập tức mà không cần F5 (Làm mới).
* 🏷️ **Đổi Tên Tự Động (Smart Rename):** Tính năng "vàng" cho Studio chụp ảnh thẻ. Chỉ cần nhập tên khách (VD: `NguyenVanA`), khi xuất ảnh phần mềm tự động đổi tên file. Nếu bôi đen nhiều ảnh, hệ thống tự động đánh số thứ tự (`NguyenVanA_1`, `NguyenVanA_2`).
* 🛡️ **Cảnh Báo Trùng Lặp (Duplicate Warning):** Ngăn chặn triệt để việc xuất nhầm 1 file 2 lần vào cùng một thư mục, giúp tiết kiệm dung lượng thẻ nhớ/ổ cứng và tránh rác dữ liệu.
* 🔄 **Chuyển Đổi Workflow Linh Hoạt:** Nút chuyển đổi nhanh giữa 2 chế độ làm việc: **Lọc Ảnh Chung** (Sự kiện/Phóng sự) và **Chụp Ảnh Thẻ** (Tối ưu cho việc đổi tên và dời file tức thì).

## ✨ Tính Năng Cốt Lõi (Core Features)

* **🚀 Phân loại Siêu tốc:** Gán sẵn 5 thư mục đích (Tương ứng phím tắt `1` đến `5`). Nhấn `Enter` để lập tức dời/copy ảnh sang các nhóm khác nhau.
* **🎛️ Hai Chế độ Xử lý:**
  * `COPY (Ctrl+C)`: Trả ảnh cho khách nhưng vẫn giữ lại bản gốc an toàn trên thẻ nhớ.
  * `CUT (Ctrl+X)`: Vừa phân loại vừa dọn dẹp thẻ nhớ ngay lập tức.
* **🎥 Hỗ trợ Đa Định dạng:** Đọc mượt mà các định dạng hình ảnh phổ thông (`JPG`, `PNG`, `WEBP`), hỗ trợ ảnh `RAW` từ máy cơ (`CR2`, `NEF`, `ARW`, `DNG`) và phát trực tiếp Video (`MP4`, `MOV`, `AVI`).
* **💽 Giám sát Thẻ nhớ An toàn:** Hiển thị dung lượng ổ đĩa/thẻ SD theo thời gian thực và tích hợp nút **Eject (Rút thẻ an toàn)** ngay trên giao diện để tránh hỏng/mất dữ liệu.
* **🔒 Bảo mật App Lock:** Khóa phần mềm bằng Master Password để bảo vệ các thư mục dự án nhạy cảm hoặc riêng tư khỏi sự tò mò của người lạ.

## ⌨️ Cẩm Nang Phím Tắt (Shortcuts)

| Phím tắt | Chức năng |
| :--- | :--- |
| `Ctrl + C` | Chuyển sang chế độ **COPY** (Sao chép) |
| `Ctrl + X` | Chuyển sang chế độ **CUT** (Di chuyển) |
| `1, 2, 3, 4, 5` | Chọn nhanh thư mục đích tương ứng (F1 -> F5) |
| `Enter` | Xuất/Dời ảnh vào thư mục đã chọn |
| `Mũi tên ⬆ / ⬇` | Lướt xem ảnh trên danh sách |
| `Space (Dấu cách)` | Phát / Tạm dừng Video |
| `Ctrl + Click chuột` | Bôi đen, chọn nhiều file cùng lúc |

## 🛠️ Công Nghệ Sử Dụng (Tech Stack)
* **Ngôn ngữ:** Python 3.x
* **Giao diện (GUI):** PyQt6
* **Xử lý Media:** PyQt6.QtMultimedia, Pillow (để đọc thông số Exif máy ảnh).
* **Kiến trúc hệ thống:** * Giao tiếp đa luồng `QThread` (Download, Cập nhật).
  * Giám sát hệ thống file `QFileSystemWatcher` (Theo dõi thời gian thực).
  * Tương tác API REST (GitHub API).
